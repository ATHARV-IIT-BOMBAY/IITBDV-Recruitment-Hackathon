## 🚀 Overview
This repository contains my final submissions and algorithm improvements for Phase 2 of the IITB Racing Driverless recruitment process. The tasks cover three core autonomous pipeline modules: Path Planning & Controls (PPC), SLAM, and Perception.

---

## 📍 1. Path Planning & Controls (PPC)
* **Lap Time:** 23.178s (Clean run, 0 penalties).
* **Upgrades Implemented:**
  * **Lateral Control:** Implemented a Pure Pursuit controller featuring dynamic lookahead for optimized cornering.
  * **Longitudinal Control:** Integrated a P-controller to ensure longitudinal speed stability throughout the track.

## 🗺️ 2. SLAM (Simultaneous Localization and Mapping)
* **Data Association:** Upgraded to the **Hungarian Algorithm** to ensure robust, global 1-to-1 feature matching.
* **Localization:** Implemented **RK4 Integration** (Runge-Kutta 4th Order) to significantly minimize trajectory drift over time.
* **Mapping:** Utilized **Recursive Mean Filtering** to refine estimated cone positions and systematically eliminate sensor noise.

## 👁️ 3. Perception
* **Task:** Monocular Distance Estimation.
* **Logic & Methodology:** Applied **Pin-hole Camera Geometry** to calculate depth. The distance is derived directly from the pixel height of the bounding boxes generated by the YOLO object detection model.
